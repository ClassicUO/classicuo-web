# Troubleshooting

Running into issues with the Web Client? You're in the right place. Pick the problem that best describes what you're
seeing:

::: tip Still stuck?

If none of the fixes on these pages solve your problem, please ask for help in the
[#web-general](https://discord.com/invite/VdyCpjQ) channel on Discord, or
[file a bug report](https://github.com/ClassicUO/classicuo-web/issues) on GitHub.

:::

## [Test your browser](/players/troubleshooting-browser-test)

Not sure if your computer can run the Web Client? Use these test pages to quickly validate your PC/Browser setup.

## [Storage errors and download failures](/players/troubleshooting-storage)

You're getting "not enough storage space" errors, game files won't download, or the client won't run in your browser at
all.

## [The game is slow, laggy, or using too much CPU](/players/troubleshooting-performance)

The game feels sluggish, your fan is spinning loudly, or you noticed high CPU usage in Task Manager.

## [ExitLag and gaming VPNs](/players/troubleshooting-gaming-vpn)

You use ExitLag or another gaming VPN and want to reduce latency or fix connection issues.

## [Using a dedicated browser for the game](/players/troubleshooting-dedicated-browser)

You use ExitLag and don't want all of Chrome routed through it, or you're on a laptop and don't want game settings
affecting your everyday browsing.
